Implemented option 3. 
FSA table is located in table.h

Assumes that comments are between the '$' symbol and cannot surpass 1 line 

To build the program type "make" on the command line.

To run the program with a file of your choice, type "./P1 yourfile" where "yourfile" is a file without the exention. ".fs" will be added as the extension within the program. 


To clean all executables and output files, type "make clean"
